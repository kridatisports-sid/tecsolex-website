# Tecsolex Website

Official website for Tecsolex — AI-native media infrastructure platform for TV & film production management.

## Deploy

### GitHub Pages
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Site goes live at `https://<username>.github.io/<repo-name>`

### Vercel (recommended)
1. Import repo at [vercel.com](https://vercel.com)
2. Framework: **Other**
3. Deploy — live in ~30 seconds

### Netlify
1. Drag & drop the folder at [netlify.com/drop](https://app.netlify.com/drop)

## Stack
- Pure HTML/CSS/JS — no build step required
- GSAP 3.12.5 (CDN)
- Google Fonts (CDN)
- Single file: `index.html`
